TimerGauge:getTime (format)
Get the amount of time remaining on the timer, in seconds

Parameters:

  • format string Format string for how to return the time. IF not provided defaults to self.timeFormat(which defaults to "S.t"). If "" is passed will return "" as the time.
    "hh" is hours with leading 0 (00 - 24)
    "h" is hours without leading 0 (0-24)
    "mm" is milliseconds with leading 0s (000 - 999)
    "m" is milliseconds without leading 0s (0-999)
    "ss" is seconds with leading 0s (00-60)
    "s" is seconds without leading 0s (0-60)
    "S" is total seconds, before breaking down into smaller units. IE if there is 60 seconds left, S will return 60 but s would return 0. (0-*)
    "t" is tenths of a second (0-9)
    "MM" is minutes with leading 0s (01 - 59)
    "M" is minutes without leading 0s (1-59)
    "dd" is days with one leading 0 (0 - ...)
    "d" is days without leading 0 (0 - ...)
