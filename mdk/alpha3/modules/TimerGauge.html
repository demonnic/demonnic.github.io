Module TimerGauge

TimerGauge Animated countdown timer, extends Geyser.Gauge

Functions

start ([show]) Starts the timergauge.
stop ([hide]) Stops the timergauge.
pause ([hide]) Alias for stop.
reset () Resets the time on the timergauge to its original value.
restart ([show]) Resets and starts the timergauge.
getTime () Get the amount of time remaining on the timer, in seconds
executeHook () Execute the timer's hook, if there is one.
finish ([skipHook]) Sets the timer's remaining time to 0, stops it, and executes the hook if one exists.
update ([skipHook]) Updates the visual representation of the gauge with time remaining
setTime (time) Sets the amount of time the timer will run for


Functions

start ([show])
Starts the timergauge. Works whether the timer is stopped or not. Does not start a timer which is already at 0

Parameters:

  • show boolean override the autoShow property. If you pass false it will not automatically pass :show() (optional)
stop ([hide])
Stops the timergauge. Works whether the timer is started or not.

Parameters:

  • hide boolean override the autoHide property. If you pass true it will ensure :hide() is called when stopped (optional)
pause ([hide])
Alias for stop.

Parameters:

  • hide boolean override the autoHide property. If you pass true it will ensure :hide() is called when stopped (optional)
reset ()
Resets the time on the timergauge to its original value. Does not alter the running state of the timer
restart ([show])
Resets and starts the timergauge.

Parameters:

  • show boolean override the autoShow property. If you pass false it will not automatically pass :show(), if you pass true it will. (optional)
getTime ()
Get the amount of time remaining on the timer, in seconds
executeHook ()
Execute the timer's hook, if there is one.
finish ([skipHook])
Sets the timer's remaining time to 0, stops it, and executes the hook if one exists.

Parameters:

  • skipHook boolean use true to have it set the timer to 0 and stop, but not execute the hook. (optional)
update ([skipHook])
Updates the visual representation of the gauge with time remaining

Parameters:

  • skipHook boolean use true if you do not want to execute the hook if the timer is at 0 (optional)
setTime (time)
Sets the amount of time the timer will run for

Parameters:

  • time number how long in seconds the timer should run for
